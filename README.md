# Breast-Tumor-Gene-Expression-Analysis-NCBI-GEO-GSE70951
This repository contains my beginner-level analysis of the NCBI GEO dataset [GSE70951], which profiles gene expression in breast adenocarcinoma and matched adjacent normal tissue. It is my first time working with an NCBI dataset, and Inshallah, I plan to extend this project with more advanced analyses in the future.
